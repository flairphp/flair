Trait CallableRuleTrait

Provides all the functionality needed to implement a Flair\Validation\Core\CallableRule and meet the requirements of the Flair\Validation\Core\RuleInterface Interface. The only thing a developer needs to do to create a rule, is create a constructor that sets the internal attributes.

Flair\Validation\Core\CallableRuleTrait uses Flair\Validation\Core\RuleTrait
