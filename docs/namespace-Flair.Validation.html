Namespace Flair\Validation

Classes summary
Replacer

The Default replace implementation. The replacer can be thought of as an advanced substr_replace.
Interfaces summary
ExceptionInterface

Allows hinting for any exception in the Flair\Validation namespace.
ReplacerInterface The blueprint to build Replacer against.
RuleInterface The blueprint to build basic rules against, and hint for all rules with.
RuleMessageInterface

The blueprint to build rules against that handle setting/getting custom error messages.
RuleReplacerInterface

The blueprint to build rules against that handle setting/getting replacer objects.
Traits summary
ReplacerTrait Provides The functionality needed to implement a Replacer.
RuleMessageTrait

Provides The minimal functionality needed to store, set, & get a rules error message/template.
RuleReplacerTrait Provides The minimal functionality needed to manipulate a replacer.
RuleTrait

Provides a partial implementation needed to meet the requirements of the Flair\Validation\RuleInterface Interface. The only thing a developer needs to do to create a rule, is implement isValid and getErrors methods.
